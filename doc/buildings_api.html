Attributes

The following building descriptor is a JSON dictionary containing properties which describe a Portland State University building.

{
  "long_name":  <string>,
  "short_name": <string>,
  "building_code": <string>,
  "building_identifier": <string>,
  "street_address": <string>,
  "city": <string>,
  "state_code": <string>,
  "zipcode": <string>,
  "geolocate_lat": <float>,
  "geolocate_long": <float>,
  "centroid_lat": <float>,
  "centroid_long": <float>,
  "rlis_lat": <float>,
  "rlis_long": <float>,
  "start_date": <string>,
  "end_date": <string>
}
  • long_name - The long name for a building, ex: “Stephen E. Epler Residence Hall”.  Must be longer than one and less than or equal to 60 unicode characters.
  • short_name - The short building name, ex: “Epler Res Hall” for the “Stephen E. Epler Residence Hall” building.  Must be longer than one and less than or equal to 30 characters.
  • building_code - The even shorter building name, ex: “SEH” for the “Epler Res Hall” building. Must be longer than one and less than or equal to 6 unicode characters.
  • building_identifier - The long-lived unique identifier for a PSU building which does not change with building name changes, ex: “B0042” is the building_identifier for the Stephen E. Epler Residence Hall.
  • street_address - The street address including street number and name, ex: “1136 SW Montgomery Street”.  Must be longer than one and less than or equal to 75 characters.
  • city - The city, ex: “Portland”
  • state_code - State, ex: “OR”
  • zipcode - The zipcode, ex: “97201”
  • location - The geographic building location as a centroid, RLIS, and Geolocate coordinates specified as a JSON floating point number element containing the latitude and longitude in decimal degrees (DDD) format, ex: { "centroid_lat": 45.512853, 'centroid_long': -122.688024 }.
  • start_date, end_date - The date range in which a building_descriptor is valid, specified as ISO-8601 date format, ex:  2015-01-07 represents January 7th, 2015.

All text fields, including building_identifiers passed as arguments, must contain only mixed case alphanumeric, comma, period, hyphen, underscore and space characters.

Actions

DELETE - Deactivate a Building

DELETE https://ws.oit.pdx.edu/org/version/buildings/building_identifier

Change the given building status to inactive in the list of current Portland State University buildings.

  • Parameters
    • version - API version
    • building_identifier - Unique building identifier.
  • Status Codes
    • 200 - Action succeeded
    • 400 - Malformed building_identifier
    • 403 - Forbidden
    • 404 - Building does not exist
  • API Versions
    • v1 - Initial release

GET - List All PSU Buildings

GET https://ws.oit.pdx.edu/org/version/buildings

Retrieve the list of current Portland State University buildings.

  • Parameters
    • version - API version
  • Status Codes
    • 200 - Action succeeded
    • 400 - Request failed
  • API Versions
    • v1 - Initial release
  • Return
    • This action returns a UTF-8 encoded JSON array of BuildingDescriptor elements

GET - Get a Building

GET https://ws.oit.pdx.edu/org/version/buildings/building_identifier

Retrieve a current Portland State University building descriptor.

  • Parameters
    • version - API version
    • building_identifier - Unique building identifier.
  • Status Codes
    • 200 - Action succeeded
    • 400 - Malformed building_identifier
    • 404 - Building does not exist
  • API Versions
    • v1 - Initial release
  • Return
    • This action returns a UTF-8 encoded JSON BuildingDescriptor

GET - Building Change History

GET https://ws.oit.pdx.edu/org/version/buildings/building_identifier/history

Retrieve a history of changes for a Portland State University building.

  • Parameters
    • version - API version
    • building_identifier - Unique building identifier.
  • Status Codes
    • 200 - Action succeeded
    • 400 - Malformed building_identifier
    • 404 - Building does not exist
  • API Versions
    • v1 - Initial release
  • Return
    • This action returns a UTF-8 encoded JSON array of BuildingDescriptor elements

POST - Create a New Building

POST https://ws.oit.pdx.edu/org/version/buildings/building_identifier/history

Creates a new Portland State University building.

  • Parameters
    • version - API version
    • building_identifier - Unique building identifier.
  • JSON Parameters
    • BuildingDescriptor - New building descriptor
  • Status Codes
    • 200 - Action succeeded
    • 400 - Request failed
    • 403 - Forbidden
    • 404 - Building descriptor is not valid. Details of descriptor errors are reported.
  • API Versions
    • v1 - Initial release
  • Return
    • This action returns a UTF-8 encoded JSON BuildingDescriptor element of the newly created building on success, otherwise nothing

  • API Versions
