Hold

https://ws.oit.pdx.edu/people/v1/hold/advise

URI Paths

  • /people/v1/hold/advise - View and clear freshman student advising holds

Actions

GET - Check if a Student has a first year advising hold applied 

					GET https://ws.oit.pdx.edu/people/version/hold/advise/check/odinid
  • Parameters
    • version - API version
    • psuid - The '9'-number of the Student to check for an advising hold
  • Status Codes
    • 200 - Action succeeded without error
    • 400 - Request failed
    • 404 - Student was not found
  • API Versions
    • v1 - Initial release
  • Required Scope
    • /people/hold/advise - Access token must be associated with the required scope
  • Returns
    • If student found and has advising hold: {"hold": "true"}
    • If student found and does not have an advising hold: {"hold": "false"}
    • If student not found: 
      {
    "error": {
        "message": "The student could not be found.",
        "type":     "person_does_not_exist",
        "cat":      "~(=^..^)"
    }
}

POST - Clear the first year advising hold by an advisor for a student 

					POST https://ws.oit.pdx.edu/people/version/hold/advise/clear/advisor_id/student_id
  • Parameters
    • version - API version
    • advisor_id - The '9'-number of the advisor clearing the advising hold.
    • student_id - The '9'-number of the student having the advising hold cleared
  • Status Codes
    • 200 - Action succeeded without error
    • 400 - Request failed
    • 404 - Advisor or student was not found
  • API Versions
    • v1 - Initial release
  • Required Scope
    • /people/hold/advise - Access token must be associated with the required scope
  • Returns
    • Advising hold cleared: {"hold": "cleared"}
    • If advisor is not authorized to clear advising holds: {"hold": "not authorized"}
    • If advisor not found: 
      {
    "error": {
        "message": "The person could not be found.",
        "type":     "person_does_not_exist",
        "cat":      "~(=^..^)"
    }
}
