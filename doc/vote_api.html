Vote

https://ws.oit.pdx.edu/people/v1/vote

URI Paths

  • /people/v1/vote - Check if the specified person is eligible to vote in student elections.

Actions

GET - Check if a Student is eligible to vote in the ASPSU Student elections

GET https://ws.oit.pdx.edu/people/version/vote/odinid
  • Parameters
    • version - API version
    • odinid - The Odin name of the Student to check for voting eligibility
  • Status Codes
    • 200 - Action succeeded without error
    • 400 - Request failed
    • 404 - Student was not found
  • API Versions
    • v1 - Initial release
  • Required Scope
    • /people/vote - Access token must be associated with the required scope
  • Returns
    • If student found and eligible to vote: {"voter": "true"}
    • If student found and not eligible to vote: {"voter": "false"}
    • If student not found: 
      {
    "error": {
        "message": "The student could not be found.",
        "type":     "person_does_not_exist",
        "cat":      "~(=^..^)"
    }
}
